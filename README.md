# -telegram-anti-cheat-bot
  این مخزن شامل کد ربات تلگرام ضد خیانت است که با استفاده از پایتون و کتابخانه telegram.ext نوشته شده. این ربات بررسی می‌کند که کاربران عضو کانال یا گروه خاصی باشند و در غیر این صورت اجازه ارسال پیام به آنها داده نمی‌شود. مناسب برای مدیریت دسترسی و افزایش امنیت گروه‌ها و کانال‌های تلگرام.
